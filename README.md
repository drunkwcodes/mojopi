# Mojo Package Index

### Installation and Usage

When cd to the project directory:

1. Use `pip install pdm`
2. `pdm install -d`
3. activate venv

    Windows:
    `.venv\Scripts\activate`

    Linux:
    `.venv/bin/activate`

4. `python src/mojopi/admin.py` to start server with admin mode.
5. Open web browser, open 127.0.0.1:5000
